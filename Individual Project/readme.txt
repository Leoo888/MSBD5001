The jupyter notebook Iidividual Project.ipynb' contains the source code of the individual project of MSBD5001 Kaggle in-class competition. 

Python 3.6 is used as the programming language in this notebook and the required packages to run the code including pandans, numpy, datetime, matplotlib, as well as sklearn. 

To generate the output, run each cell one by one in the notebook. The 'submission_random_forest2.csv' and 'submission_random_forest_tuned.csv' are the two final results submission csv files.   

'train.csv' contains the training data with play time and 'test.csv' contains testing data without playtime column. 'samplesubmission.csv' are the sample submission exaple downloaded from Kaggle without any prediction results.
